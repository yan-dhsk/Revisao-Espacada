# Revisao-Espacada
O projeto descrito trata-se de um sistema de revisão para estudos. Ele utiliza dos princípios de repetição espaçada para relembrar você dos dias que você deve revisar algo estudado. Salva todos os dados necessários em um banco de dados através do SQLite. Utiliza também, uma interface gráfica através do Tkinter para interação com o usuário.
